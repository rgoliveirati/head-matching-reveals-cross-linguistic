```latex
We verified that the choice of direction does not change the qualitative picture: under \textit{dep\_to\_head}, the index-based $\rho_{\text{micro}}$ values are $0.14$, $0.11$, $0.11$, and $0.07$ for \textit{nsubj}, \textit{obj}, \textit{case}, and \textit{amod}, respectively, while the matched LOO correlations remain in the moderate range ($0.47$--$0.57$, depending on the relation).
```

## Claims requiring qualification

1. The statement that removing French changes aggregate metrics by `≤0.014` is correct for matched LOO, but not for all metrics. In `without_french_control.csv`, index-based and macro values can shift more. Recommended: say `changes the aggregate matched LOO correlations by ≤0.014`.
2. The statement that train/test profiles correlate at `ρ≈0.98–0.99` is not directly supported by the exported `train_test_stability.csv`, which reports metric deltas rather than profile-level correlations. Keep this only if a separate CSV/notebook output documents the profile-level correlation. Otherwise replace it with the matched-LOO delta statement.
3. The alternative-direction values should be updated, especially `nsubj`: exported value is `0.14`, not `0.08`.
